# Universal Thread Flow Template

## ‚ùå WRONG: Independent Points on Same Topic

```
Tweet 1: "MYTH: Cold exposure is brave..."
Tweet 2: "Cold showers = misery? Wrong!..."
Tweet 3: "Think cold exposure is TORTURE?..."
```

**Problem:** 3 separate points, no connection, no flow

---

## ‚úÖ RIGHT: Progressive Narrative Flow

```
Tweet 1: "MYTH: Cold exposure is just for the brave... [HOOK]"
Tweet 2: "Here's what actually happens when you hit that cold water: Your body releases norepinephrine, cortisol regulation kicks in... [MECHANISM]"
Tweet 3: "Start simple: 30 seconds cold at shower end. Build to 2 minutes over 2 weeks. Track your energy levels... [ACTION]"
```

**Solution:** Progressive story - Hook ‚Üí Science ‚Üí Action

---

## üìù Required Thread Flow Patterns:

### **Pattern 1: Problem ‚Üí Science ‚Üí Solution**
```
1. "Most people think X is Y..." [Problem/Myth]
2. "But here's what research shows: [mechanism]..." [Science]
3. "Here's how to apply it: [steps]..." [Solution]
```

### **Pattern 2: Question ‚Üí Answer ‚Üí Deeper Insight**
```
1. "Why does X happen?..." [Question]
2. "The answer: [mechanism]..." [Answer]  
3. "But here's what they don't tell you: [twist]..." [Deeper layer]
```

### **Pattern 3: Claim ‚Üí Evidence ‚Üí Action**
```
1. "X can change Y in Z days..." [Bold claim]
2. "Study from [source] found [data]..." [Evidence]
3. "Protocol: Do X, measure Y, adjust Z..." [Actionable]
```

---

## üîó Connecting Words (MANDATORY):

**Tweet 2 must start with:**
- "Here's why:"
- "The science:"
- "What actually happens:"
- "But here's the truth:"
- "The mechanism:"

**Tweet 3 must start with:**
- "Here's how to use it:"
- "The protocol:"
- "What this means for you:"
- "The actionable part:"
- "So what should you do:"

**Tweet 4+ must start with:**
- "The results:"
- "The takeaway:"
- "One more thing:"
- "The catch:"

---

## üö´ BANNED: Standalone Statements

**Don't:**
- Make each tweet a complete, independent thought
- Repeat the same point 3 different ways
- Use different hooks for each tweet
- Start each tweet with "MYTH:" or emoji without connection

**Do:**
- Build on previous tweet
- Use connecting phrases
- Progress the narrative
- Create anticipation for next tweet

---
